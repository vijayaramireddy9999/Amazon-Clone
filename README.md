# 🛒 Amazon Clone Landing Page

Welcome to my **Amazon Landing Page Clone**!  
This project is a **frontend practice website** built using pure **HTML5**, **CSS3**, and **Font Awesome** for icons.  
It replicates the basic structure and design of the Amazon homepage to strengthen my web development skills and give me **hands-on practical experience** with **HTML and CSS**.

---

## 📖 About the Project

This is a **static clone** of Amazon's homepage. It demonstrates:
- Building a clean HTML structure.
- Styling with CSS Flexbox/Grid.
- Using external icon libraries like Font Awesome.
- Organizing code for readability and easy updates.

This project is for **educational purposes only** and is **not affiliated with Amazon**.

---

## ✨ Features

✅ Fully structured **navigation bar** with logo, location, search box, account, returns & orders, and cart.  
✅ Hero section with a promotional message and background.  
✅ Multiple **product category boxes** with background images.  
✅ Simple, clean **footer** with company links and info.  
✅ Organized HTML and CSS for easy understanding.
